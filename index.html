How this log works:
Every time someone opens this dashboard and enters their name, it's recorded here with their timestamp and which tabs they visited.
The log is stored in the browser on each device — so if someone views on a different computer, their entry appears there.
To see all viewers across all devices, collect the log from each device.

How to keep this dashboard current every month

1
Update your Google Sheet — add new exits to the Exit Data tab and update the Active Workforce tab (remove the person who left). Keep the same column structure.
2
Download as Excel — in Google Sheets go to File → Download → Microsoft Excel (.xlsx). Save it somewhere easy to find.
3
Open this HTML file in any browser (Chrome, Edge, Firefox). Click "Update Data" in the top right corner.
4
Select the downloaded Excel file — the dashboard will instantly update with all new data. Every chart, KPI, and table refreshes automatically.
5
That's it. No internet needed after initial load. No subscriptions.
